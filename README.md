# 1000万円の保険金 受取の経緯（スマートゥーン版）

静香おばさんからかけていただいた養老保険1000万円の受取に至るまでの経緯を、
スマートフォン向け縦スクロール漫画（スマートゥーン形式）でまとめたものです。

全6章構成。2015年から2026年までの約10年間の出来事を時系列で記録しています。

---

## 📁 ファイル構成

```
manga-insurance/
├── index.html              ← メインページ
├── README.md               ← このファイル
└── images/                 ← 漫画画像（WEBP形式）を配置するフォルダ
    ├── ch01-hajimari.webp        第1章 保険のはじまり（2015年）
    ├── ch02-meigi-henkou.webp    第2章 名義変更（2022年）
    ├── ch03-gokai.webp           第3章 誤解の発生（2024年）
    ├── ch04-modosu.webp          第4章 名義を戻す（2025年前半）
    ├── ch05-tetsuduki.webp       第5章 静香おばさんの死と手続き（2025年後半）
    └── ch06-furikaeri.webp       第6章 利文兄さんとの会話、振り返り（2026年）
```

## 🚀 公開方法（GitHub Pages）

### 1. リポジトリ作成
GitHub上で新しいリポジトリを作成します（例: `manga-insurance`）。

### 2. ファイルをアップロード
このフォルダ内のすべてのファイル（`index.html`, `README.md`, `images/` フォルダ）をリポジトリにpushします。

```bash
git init
git add .
git commit -m "Initial commit: 保険金受取の経緯 漫画"
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/manga-insurance.git
git push -u origin main
```

### 3. GitHub Pages を有効化
1. リポジトリの **Settings → Pages** に移動
2. **Source** を `Deploy from a branch` に設定
3. **Branch** を `main` / `(root)` に設定して保存
4. 数分後、`https://あなたのユーザー名.github.io/manga-insurance/` で公開されます

## 🖼️ 画像について

- **形式**: WEBP（軽量・高品質）
- **推奨幅**: 720〜1080px（スマホで見やすい縦長サイズ）
- **配置**: `images/` フォルダ内に、上記の正確なファイル名で配置してください
- ファイル名を変更したい場合は、`index.html` 内の該当の `src=` 部分も合わせて書き換えてください
- 画像が未配置でもページは表示されます（プレースホルダーが出ます）

## 🎨 ページの特徴

- 📱 **モバイル最適化**: スマートフォンでの縦スクロール閲覧に最適化
- 📖 **目次機能**: 右上の「目次」ボタンから各章へジャンプ可能
- 📊 **読書進捗バー**: ページ上部に閲覧の進み具合を表示
- ⬆️ **トップへ戻るボタン**: スクロール後に画面右下に出現
- 🪶 **和紙風デザイン**: 落ち着いた和の色調、明朝体の見出し
- ⚡ **軽量**: 外部依存はGoogle Fontsのみ。シンプル構成で高速表示
